# 📝 Do Note — Flutter App

A beautiful, fully functional task manager app with calendar integration.

## Features

- 📅 **Calendar Tab** — Interactive monthly calendar, upcoming tasks view, summary cards
- ✅ **Tasks Tab** — Add/complete/delete tasks, progress ring, category badges, due date & time
- ⚙️ **Settings Tab** — Dark/Light mode toggle, Language selector (8 languages), About page
- 🌙 **Dark & Light Mode** — Fully themed across all screens
- 📝 **Add Task Sheet** — Date picker, time picker, category selector

## How to Run

```bash
# 1. Create a new Flutter project
flutter create do_note
cd do_note

# 2. Replace pubspec.yaml with the provided one

# 3. Replace lib/main.dart with the provided file

# 4. Install dependencies
flutter pub get

# 5. Run
flutter run
```

## Tech Stack
- Flutter 3.x
- Material 3
- Google Fonts (Nunito + Quicksand)
- Custom painters for progress ring
- AnimatedContainer for smooth transitions
